# Солнце Москвы — ТЗ для оценки работ

---

## О проекте

**Клиент:** МФК «Солнце Москвы» — многофункциональный комплекс с крупнейшим в Европе колесом обозрения (140 м), расположен у южного входа ВДНХ.

**Задача:** Редизайн основного сайта moscow-sun.ru + редизайн интерфейса билетного сервиса tickets.moscow-sun.ru (без переработки функционала — только визуал).

**Текущая платформа:** 1С-Битрикс. Билетный сервис — отдельный поддомен у стороннего подрядчика.

---

## Что услышали от клиента

- Работают с 2022 года, текущий сайт устарел визуально и структурно
- Предыдущий подрядчик работал с осени 2025 — не договорились даже на уровне прототипов. Коммуникация не сложилась
- Пришли по рекомендации от проекта «Бани Малеевича»
- Сроки: **3–6 месяцев**, к летнему сезону не спешат, но интересует стоимость ускоренной разработки
- Ориентировочный бюджет: **1,5 млн ₽** — без интерактивной карты и без дизайна билетного сервиса
- Целевая аудитория клиентом не уточнялась — определена нами в ходе исследования
- Ключевые сущности: колесо обозрения, арендаторы МФК (~37 объектов), билетный сервис, интерактивная карта, мероприятия

**Риски:**
- Зависимость от стороннего подрядчика билетного сервиса (координация, сроки)
- Высокие ожидания к коммуникации (негативный опыт с предыдущим подрядчиком)
- Интерактивная карта — отдельный объём, не входит в базовый бюджет

### Ключевые находки (кратко)

1. **SEO-балл текущего сайта: 34 из 100** — нет meta description, нет H1, ноль Schema-разметки, засорённый sitemap
2. **4 разных версии графика работы** на одном сайте — прямое топливо для негатива
3. **Билетный сервис визуально оторван** от основного сайта (разный дизайн, разная навигация, разный логотип)
4. **Мероприятия** (свадьбы, корпоративы, свидания) — направление с самым высоким чеком, но слабо упаковано
5. **Рейтинг 5.0 на Яндекс Картах** (20 000+ оценок) — мощный актив, не используется на сайте
6. **Реселлеры продают дешевле** официального сайта — нужно конкурировать за собственные продажи
7. **Нет ключевых страниц:** «О комплексе», «Планируйте визит», «Парковка», комбо-билеты, подарочные сертификаты

## Предлагаемая карта сайта

### Основная навигация

```
Колесо | Билеты и цены | Рестораны и МФК | Мероприятия | Планируйте визит | О комплексе
                                                                    [CTA: Купить билет]
```

### Структура (сводка)

```
moscow-sun.ru

├── Колесо обозрения

│   ├── Типы кабинок
│   └── Что видно с высоты
│
├── Билеты и цены
│   ├── Тарифная сетка
│   ├── Комбо-билеты
│   ├── Подарочные сертификаты
│   ├── Групповые посещения
│   └── Виджет покупки
│
├── Рестораны и МФК
│   ├── Каталог арендаторов
│   ├── Карточка арендатора (шаблон × ~37)
│   ├── Интерактивная карта
│   └── Городской променад
│
├── Мероприятия
│   ├── Обзорная страница
│   ├── 8–10 типов мероприятий (шаблон)
│   └── Форма заявки
│
├── Планируйте визит
│   ├── Как добраться
│   ├── Парковка
│   ├── Режим работы
│   ├── Правила посещения
│   ├── Советы
│   ├── Доступная среда
│   └── FAQ
│
├── Афиша (события + акции)
├── Новости
├── О комплексе (история, факты, СМИ, партнёры)
```

## Новые страницы (которых сейчас нет на сайте)

| Страница | Зачем | Приоритет |
|----------|-------|-----------|
| **Статус работы колеса** (виджет на главной) | Люди приезжают и уходят из-за погоды — #1 источник негатива | Критично |
| **Планируйте визит** (хаб) | Вся практическая инфо в одном месте, сейчас разбросана по футеру | Критично |
| **Парковка** | Полностью отсутствует на сайте, частый вопрос | Критично |
| **Режим работы** (единая страница) | 4 разных версии на сайте — нужен единый источник | Критично |
| **О комплексе** | Нет имиджевой страницы с историей, фактами, архитектурой | Важно |
| **Типы кабинок** | Стандарт vs прозрачный пол — нет сравнения, люди жалуются на переплату | Важно |
| **Что видно с высоты** | Фотогид/панорама — конверсионный контент | Важно |
| **Комбо-билеты** | ВДНХ + Москвариум + экскурсии — неиспользованная возможность | Важно |
| **Подарочные сертификаты** | Все мировые аналоги продают, у нас нет | Важно |
| **Советы перед посещением** | Когда ехать, что взять, как сэкономить — снижает нагрузку на поддержку | Рекомендация |
| **Доступная среда** | Нет информации для маломобильных групп | Рекомендация |
| **Блог / Журнал** | SEO-трафик по целевым запросам | Рекомендация |
| **Для бизнеса** (хаб) | B2B-страница: аренда, реклама, корпоративы — сейчас в футере | Рекомендация |
| **Предложение руки и сердца** | Отдельный сценарий от «свидания» — высокий чек | Рекомендация |
| **Фотосессия** (тип мероприятия) | Есть в услугах, нет отдельной страницы | Рекомендация |

## Что объединить / убрать

| Действие | Что | Причина |
|----------|-----|---------|
| Убрать | Страница «Ковид-19» | Неактуальна, битый роутинг |
| Объединить | «Тарифы» + «Билеты и цены» | Дублирование |
| Объединить | Правила МФК + Правила аттракциона | Два юр. текста → одна человеческая страница + ссылки на полные версии |
| Убрать | «Правила фотоконкурса» | Если неактуален |
| Объединить | «Способы оплаты» + возвраты | Сейчас возвраты спрятаны на странице оплаты |

## Что доработать на существующих страницах

| Страница | Что сделать |
|----------|------------|
| **Главная** | Hero с видео + CTA, блок отзывов, статус колеса, тарифы кратко, акции |
| **Колесо обозрения** | Добавить факты/рекорды, видео из кабинки, разбивку по типам кабинок |
| **Рестораны и магазины** | Улучшить фильтрацию (кухня, чек, часы), стандартизировать карточки |
| **Интерактивная карта** | Полная переработка (по запросу клиента), текстовый слой для SEO |
| **Мероприятия** | Каждый тип = продуктовая страница (галерея, цена, что входит, FAQ) |
| **FAQ** | Расширить, вынести из футера в основную навигацию |
| **Карточки арендаторов** | Стандарт: название, описание, этаж, часы, кухня, чек, фото, ссылка |

## Отдельные работы

| Работа | Описание |

| **Редизайн билетного сервиса** | Только визуал: единая типографика, цвета, навигация с основным сайтом. Координация с подрядчиком |
